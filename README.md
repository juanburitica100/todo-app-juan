# Mi primer to-do app

Esta es una aplicación sencilla para gestionar tareas (to-dos) creada como parte de un proyecto educativo. Permite a los usuarios agregar, completar, archivar y filtrar tareas de manera eficiente.

## Características

- Agregar nuevas tareas.
- Marcar tareas como completadas.
- Archivar tareas completadas.
- Filtrar tareas por estado: todas, abiertas, cerradas y archivadas.

## TechStack

- HTML5
- CSS3
- JavaScript (ES6+)
- LocalStorage para persistencia de datos
- Tailwind CSS para estilos rápidos y responsivos
